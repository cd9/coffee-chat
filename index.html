Coffee Chat

Usage:

1. Input names into List of People. One name per line.

2. Input comma-seperated pairs of past pairs into Table of Past Meetings. Follow the example format (exporting a spreadsheet to CSV matches this format).

3. Click generate to generate a unique list of pairs

Notes:

- This tool does not support triplets. If the list of names is odd then the extra person will be appended onto the first pair.
