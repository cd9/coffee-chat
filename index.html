Round Robin Scheduler

Usage:

1. Input names into List of People. One name per line.

2. Input comma-seperated list of banned pairings into Table of Banned Pairings.
Follow the example format (exporting a spreadsheet to CSV matches this format).

3. Click generate to generate a round robin schedule.

If the list of people is odd, one person will remain unpaired per round.
